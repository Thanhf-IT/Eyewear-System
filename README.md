# Eyewear-System

Dự án được tách thành 2 ứng dụng độc lập để dễ dàng phát triển và gỡ lỗi:

- `backend/`: API viết bằng PHP, xử lý logic nghiệp vụ và làm việc với cơ sở dữ liệu MySQL.
- `frontend/`: Giao diện người dùng sử dụng HTML/CSS/JavaScript.
